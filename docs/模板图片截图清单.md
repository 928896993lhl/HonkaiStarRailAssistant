## 🛠 截图实操 Tips

1. **打开星穹铁道 PC 端** → 设置 → 图形 → 窗口模式 → 分辨率选 `1920×1080`
2. **用 Windows 自带的截图工具** `Win + Shift + S`，框选按钮区域
3. **粘贴到画图/PS** → 裁剪到按钮本身的边界，不要留太多边距
4. **保存为 PNG** → 放到对应的 `image/` 子目录
5. **运行验证**：`python scripts/check_resource.py` 确认缺失数降到 0

### 推荐的截图顺序

```
先截 common/ (4张) → 进游戏截 login/ (3张) → 
截 combat/ (8张) → 截 daily/ (5张) → 
截 dispatch/ (4张) → 截 weekly/ (2张) → 
截 universe/ (8张)
```

按这个顺序走一遍游戏流程，自然就能截全。
